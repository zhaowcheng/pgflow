# Rocky Linux 9 PostgreSQL Builder Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增一个支持 PostgreSQL 14+ 执行 `make world` 和 `make check-world` 的 Rocky Linux 9 通用编译镜像。

**Architecture:** 单个 Dockerfile 从指定的内网 Rocky Linux 9 镜像派生，通过 CRB 和 Development Tools 提供完整构建工具与依赖。镜像不包含 PostgreSQL 源码，最终以普通用户 `hiflow` 和 `/home/hiflow` 工作目录启动。

**Tech Stack:** Dockerfile、Rocky Linux 9、DNF、GCC、PostgreSQL 构建依赖

## Global Constraints

- 基础镜像必须是 `192.168.95.251:8082/hiflow-docker-group/library/rockylinux:9`。
- 支持 PostgreSQL 14 及以上版本的 `make world` 和 `make check-world`。
- 不下载、不固定或编译任何 PostgreSQL 源码。
- 默认 locale 为 `en_US.UTF-8`。
- 默认用户为无 sudo 权限的普通用户 `hiflow`。
- 默认工作目录为 `/home/hiflow`。
- 按用户要求不执行静态检查、镜像构建或 PostgreSQL 编译测试。

### Task 1: 新增 EL9 编译镜像

**Files:**
- Create: `builders/Dockerfile.builder-el9`

**Interfaces:**
- Consumes: Rocky Linux 9 的 BaseOS、AppStream 和 CRB 软件仓库。
- Produces: 默认以 `hiflow` 用户在 `/home/hiflow` 运行 Bash 的 PostgreSQL 编译环境。

- [ ] **Step 1: 创建 Dockerfile**

写入以下完整内容：

```dockerfile
FROM 192.168.95.251:8082/hiflow-docker-group/library/rockylinux:9

# 安装 PostgreSQL 完整编译、文档生成和回归测试所需的工具及开发依赖。
# CRB 提供部分在 Rocky Linux 9 默认仓库中不可用的开发包。
RUN dnf -y install dnf-plugins-core && \
    dnf config-manager --set-enabled crb && \
    dnf -y group install "Development Tools" && \
    dnf -y install \
        autoconf \
        automake \
        bison \
        bzip2 \
        clang-devel \
        diffutils \
        docbook-dtds \
        docbook-style-xsl \
        file \
        findutils \
        flex \
        gettext-devel \
        git \
        glibc-langpack-en \
        gzip \
        hostname \
        krb5-devel \
        libedit-devel \
        libicu-devel \
        libselinux-devel \
        libtool \
        liburing-devel \
        libuuid-devel \
        libxml2 \
        libxml2-devel \
        libxslt \
        libxslt-devel \
        libzstd-devel \
        llvm-devel \
        lz4-devel \
        openjade \
        openldap-devel \
        openssh-clients \
        openssl-devel \
        pam-devel \
        patch \
        perl \
        perl-devel \
        perl-ExtUtils-Embed \
        perl-IPC-Run \
        perl-Test-Harness \
        perl-Test-Simple \
        perl-Time-HiRes \
        pkgconf-pkg-config \
        procps-ng \
        python3 \
        python3-devel \
        readline-devel \
        rsync \
        shadow-utils \
        systemd-devel \
        systemtap-sdt-devel \
        tar \
        tcl-devel \
        unzip \
        wget \
        which \
        xz \
        zlib-devel && \
    dnf clean all && \
    rm -rf /var/cache/dnf

# PostgreSQL 初始化和回归测试默认使用 UTF-8 locale。
ENV LANG=en_US.UTF-8 \
    LC_ALL=en_US.UTF-8

# 使用无 sudo 权限的普通用户进行源码编译和测试。
RUN useradd --create-home --shell /bin/bash hiflow

USER hiflow
WORKDIR /home/hiflow

CMD ["/bin/bash"]
```
